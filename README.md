## Installation

To get a local copy up and running, follow these simple steps:

1. Clone the repository:
   ```sh
   git clone https://github.com/ashishk15678/gdg-niet.git
   ```
2. Navigate to the project directory:
   ```sh
   cd gdg-niet
   ```
3. Install dependencies:
   ```sh
   npm install
   ```
4. Run the project:
   ```sh
   npm run dev
   ```
